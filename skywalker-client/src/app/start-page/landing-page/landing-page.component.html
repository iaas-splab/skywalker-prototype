What you may do first:
Adding Templates and Mapping Modules

For generating Application Models, you need at least one Template and a suitable Mapping Module.

Analyze Application Models

Subsequently, you can find the existing Application Models at Applications view.

Evaluate portability

Open the desired Application to analyze the incorporated services which are displayed with generic descriptive names. In addition, click Evaluate at the bottom of the detail view to evaluate the portability to a chosen provider platform, e.g., Azure.

Maintain Mapping Repositories

In Applications, there are generic Service and Property names for provider-specific resources. In Service Repo and Property Repo you can look up how the mapping is done and update the mapping tables, if necessary.
